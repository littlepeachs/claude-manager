# 🎉 Dashboard 更新说明 v2.0

## ✨ 新功能

### 1. 改进的会话创建界面

点击右下角 **+** 按钮，现在会弹出一个完整的创建会话对话框：

#### 📁 路径选择方式

**选项 1: 新建文件夹**
- 输入新文件夹名称
- 自动在 `/Users/liwentao/Desktop/Research/claude/` 下创建
- 例如输入 `my-project`，会创建 `/Users/liwentao/Desktop/Research/claude/my-project`

**选项 2: 选择现有文件夹**
- 从列表中选择已存在的文件夹
- 显示所有可用的项目文件夹
- 点击选中，再次点击取消选中

#### 🎯 Claude 模式选择

支持三种模式：

1. **Ask 模式** (默认)
   - 标准问答模式
   - 适合一般对话和咨询
   - 命令: `claude`

2. **AutoEdit 模式**
   - 自动编辑模式
   - Claude 会自动修改文件
   - 命令: `claude --autoedit`

3. **Plan 模式**
   - 规划模式
   - 先规划再执行
   - 命令: `claude --plan`

---

## 🔧 使用步骤

### 创建新会话

1. **点击右下角 + 按钮**

2. **选择路径方式**
   - 新建文件夹：输入名称如 `test-project`
   - 选择现有：从列表中点击选择

3. **选择 Claude 模式**
   - 点击 Ask / AutoEdit / Plan 其中一个

4. **点击"创建会话"**
   - 系统会自动创建 tmux 会话
   - 如果是新文件夹，会自动创建目录
   - 会话会出现在 Dashboard 卡片中

5. **开始使用**
   - 点击新创建的会话卡片激活它
   - 在顶部输入框输入问题
   - 点击发送或按 Enter

## 🔄 模式说明

### Ask 模式
- **用途**: 一般问答、咨询、学习
- **特点**: Claude 不会自动修改文件
- **适合**: 代码审查、问题咨询、学习讨论

### AutoEdit 模式
- **用途**: 快速开发、代码修改
- **特点**: Claude 会自动编辑文件
- **适合**: 快速迭代、批量修改、重构

### Plan 模式
- **用途**: 复杂任务、多步骤操作
- **特点**: 先制定计划，用户确认后执行
- **适合**: 大型重构、架构设计、复杂功能

## ⚠️ 注意事项

### 1. 文件夹创建

- 新建文件夹会自动创建目录
- 如果目录已存在，会直接使用
- 确保有写入权限

### 2. 模式切换

- 创建会话时选择的模式是固定的
- 如需切换模式，需要创建新会话
- 或在终端手动切换

### 3. 会话管理

- 每个会话独立运行
- 可以同时运行多个不同模式的会话
- 建议为不同任务创建不同会话

## 🐛 故障排除

### 问题 1: 创建失败

**可能原因**:
- 文件夹名称包含特殊字符
- 没有写入权限
- tmux 未正确安装

**解决方法**:
```bash
# 检查权限
ls -la /Users/liwentao/Desktop/Research/claude/

# 手动创建测试
mkdir /Users/liwentao/Desktop/Research/claude/test
tmux new-session -s test "cd /Users/liwentao/Desktop/Research/claude/test && claude"
```

### 问题 2: 文件夹列表为空

**可能原因**:
- 基础路径不存在
- 权限问题

**解决方法**:
```bash
# 检查目录
ls /Users/liwentao/Desktop/Research/claude/

# 刷新页面重新加载
```

### 问题 3: 模式不生效

**检查**:
- 查看会话是否正确启动
- 连接到 tmux 会话查看: `tmux attach -t SESSION_NAME`
- 检查 Claude 命令是否正确

## 📊 API 更新

### 新增端点

**GET /api/folders**
- 获取可用的项目文件夹列表
- 返回: `{"folders": ["folder1", "folder2", ...]}`

**POST /api/sessions/create**
- 创建新会话（已更新）
- 参数:
  ```json
  {
    "project_path": "/path/to/project",
    "mode": "ask|autoedit|plan"
  }
  ```
